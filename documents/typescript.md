# Typescript

## Instalar typescript

en node.js
```bash
	npm install -g typescript			  Instalar typescript
	tsc -v								 Versión Instalada 
	tcs namefile.ts						Compilar TS a JS
	tsc -w *.ts							Escucha modificaciones de archivos .ts (compilación automática)
```
